# Tech-Talk Pattern (10-15 slides)

技術発表・勉強会向け。

## Structure

```
1. Cover (cover)
   - タイトル、発表者名

2. Agenda (agenda)
   - 3-5項目の目次

3. Background/Context
   - なぜこの話をするのか
   - 前提知識・状況説明

4-5. Problem Statement
   - 解決したい課題
   - 具体的な困りごと

6-8. Solution/Approach (two-col, flow)
   - 採用したアプローチ
   - 技術的な詳細
   - コード例

9-10. Results/Demo (kpi-cards)
   - 実装結果
   - デモ・スクリーンショット

11-12. Lessons Learned
   - 得られた知見
   - 注意点・Tips

13. Summary (lead)
   - 要点まとめ（3点以内）

14. Next Steps
   - 今後の展望
   - 聴衆へのアクション

15. Q&A / Closing (closing)
    - 連絡先・リンク
```
