# Short Video Hashtag Strategy (Multi-Platform)

## Platform Comparison

| Platform | Max Tags | Recommended | Placement | Special |
|----------|----------|-------------|-----------|---------|
| Instagram | 30 | 15-20 | First comment (recommended) | Separated by dot spacers in caption |
| YouTube Shorts | 15 | 3-5 | Description end | `#Shorts` mandatory |
| TikTok | ~5 | 3-5 | Inline in caption | Fewer = better reach |

## Instagram Strategy

### Count: 15-20 tags
### Placement: First Comment (recommended)

**Caption**: Clean caption without hashtags (or 3-5 blended naturally)
**First Comment**: 15-20 additional hashtags

### Tier Distribution
| Tier | Volume | Count | Purpose |
|------|--------|-------|---------|
| Mega | 1M+ | 2-3 | Broad reach |
| Mid | 100K-1M | 5-7 | Target audience |
| Niche | 10K-100K | 5-7 | Community |
| Brand | Any | 1-3 | Identity |

## YouTube Shorts Strategy

### Count: 3-5 tags + `#Shorts`
### Placement: End of description

**Rules**:
- `#Shorts` is always required (last hashtag)
- Max 3-5 additional relevant hashtags
- Focus on searchability (YouTube is a search engine)
- Use exact-match keywords people search for
- No hashtags in title

### Template
```
{description}

#keyword1 #keyword2 #keyword3 #Shorts
```

### Examples (Japanese)
```
#シフト管理 #業務効率化 #飲食店 #Shorts
#DX推進 #人手不足解消 #SaaS #Shorts
#店長の味方 #自動シフト #ShiftBud #Shorts
```

## TikTok Strategy

### Count: 3-5 tags
### Placement: Inline at end of caption

**Rules**:
- Fewer hashtags = better reach (TikTok algorithm preference)
- Mix trending + niche + brand
- Hashtags are part of the caption (count toward 2,200 char limit)
- Check TikTok Discover for trending tags
- Avoid banned/restricted hashtags

### Template
```
{caption text}

#tag1 #tag2 #tag3 #brand
```

### Examples (Japanese)
```
#シフト管理 #飲食店 #業務効率化 #ShiftBud
#店長あるある #バイト管理 #DX #playpark
#人手不足 #働き方改革 #AIシフト #ShiftBud
```

## Generation Rules (Cross-Platform)

1. **Extract keywords** from content/topic
2. **Select per-platform** tag sets respecting count limits
3. **Always include brand tags** (`#ShiftBud`, `#playpark`)
4. **YouTube**: Always append `#Shorts`
5. **TikTok**: Keep to 3-5 max, prioritize trending
6. **Instagram**: Full tier distribution (15-20)
7. **Mix languages** for bilingual accounts
8. **Check platform-specific banned tags**
