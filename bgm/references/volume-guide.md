# 音量バランスガイド

## 基本原則

**ナレーション明瞭性最優先**: BGM は「背景」として存在し、ナレーションを邪魔してはならない。
人声の基本周波数(85-255Hz + 倍音)と BGM が重なる帯域には特に注意。

## 音量設定表

| 音声タイプ | 相対音量 | 説明 |
|-----------|---------|------|
| ナレーション | 1.0 (100%) | メイン音声、原音量維持 |
| BGM（ナレーション密） | **0.05-0.08** | ナレーションの明瞭性確保 |
| BGM（ナレーション疎） | 0.08-0.12 | ナレーション間隔が多い場合 |
| BGM（ナレーション無し区間） | 0.15-0.25 | オープニング/エンディング/トランジション |
| BGM（音楽のみ区間） | 0.30-0.50 | ナレーション無し時 |

## 音量選択フロー

```
動画にナレーションがあるか？
├── あり → ナレーション密度が高いか（>70% にナレーション）？
│   ├── 高い → BGM volume = 0.05-0.06（推奨 0.05）
│   └── 低い → BGM volume = 0.08-0.10
└── なし → BGM volume = 0.25-0.40
```

## テスト方法

本番レンダリング前に、ナレーションを含む 30 秒を抜き出してテスト:

```bash
for vol in 0.04 0.05 0.06 0.08; do
  ffmpeg -y -i bgm.mp3 -i voiceover.mp3 \
    -filter_complex "[0:a]volume=$vol[bgm];[1:a][bgm]amix=inputs=2[out]" \
    -map "[out]" -t 30 test_vol_$vol.mp3
done
```

## シーン別動的音量の例

| シーン | 音量 | 理由 |
|--------|------|------|
| オープニング (0-8s) | 0.15 | ナレーション無し |
| 本編 (8-72s) | 0.05 | ナレーション密 |
| エンディング (72-85s) | 0.12 | ナレーション疎 |
