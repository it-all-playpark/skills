# Instagram Caption Guide

## Caption Limits

| Type | Limit | Visible Before Fold |
|------|-------|---------------------|
| Feed/Carousel | 2,200 chars | First 125 chars |
| Reel | 2,200 chars | First 125 chars |
| Story | Not displayed | N/A |

## Media Requirements

### Images
- Formats: JPEG, PNG
- Max size: 8 MB
- Recommended: 1080 x 1350 px (portrait 4:5, best engagement)
- Square: 1080 x 1080 px
- Landscape: 1080 x 566 px (1.91:1)
- Vertical: 1080 x 1920 px (9:16, Stories/Reels only)

### Videos
- Formats: MP4, MOV
- Codec: H.264, 30 fps
- Width: 1080 px
- Feed: max 300 MB, max 60 min
- Reel: max 300 MB, max 90 sec, min 3 sec, 9:16 aspect
- Story: max 100 MB, max 60 sec

### Carousel
- Up to 10 items (mixed image/video)
- First item determines aspect ratio for all

## Caption Writing Principles

### First 125 Characters (Critical)
- Most important content here (visible before "more" fold)
- Start with hook: question, bold statement, or surprising fact
- No hashtags in this section
- Include primary keyword naturally

### Body
- Break into short paragraphs (2-3 sentences)
- Use line breaks (empty lines) for readability
- Add value: tips, insights, behind-the-scenes, stories
- Include a clear CTA (call-to-action)

### CTA Examples (Japanese)
- `保存してね` (Save this!)
- `コメントで教えて` (Tell us in comments)
- `プロフィールのリンクから詳細をチェック` (Check link in bio)
- `この投稿が役立ったらシェアしてね` (Share if helpful)
- `フォローして最新情報をゲット` (Follow for updates)

## Content Type Templates

### Feed - Japanese

```
{フック - 問いかけor驚き}

{価値提供 - 3-5行}

{CTA}

---
💡 プロフィールのリンクから詳細をチェック

{ハッシュタグ}
```

### Reel - Japanese

```
{短いフック - 1行}

{動画の内容要約 - 2-3行}

{CTA}

{ハッシュタグ}
```

### Carousel - Japanese

```
{テーマ紹介 - スワイプ促進}

📸 スワイプして{N}つのポイントをチェック →

{各スライドの簡単な説明}

{CTA - 保存推奨}

{ハッシュタグ}
```

## Instagram SEO Signals

1. **Alt text**: Describe image content for accessibility + SEO
2. **Caption keywords**: Natural keyword placement in first 2 lines
3. **Hashtags**: Mix of sizes (see hashtag-strategy.md)
4. **Location tag**: Add when relevant
5. **Engagement**: Prompt saves and shares (algorithm signals)
